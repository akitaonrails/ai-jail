# v1.4.1

## DNS, dry-run, and packaging cleanup

### Fixed

- Fixed DNS resolution on systemd-resolved + Tailscale split-DNS systems (#49).
  When the host resolver uses the `127.0.0.53` stub and the upstream resolver list contains tunnel-style DNS servers, ai-jail now keeps the stub resolver so systemd-resolved can preserve its per-domain routing.
- Made `--dry-run` read-only: it no longer creates or updates project `.ai-jail` config, and it no longer persists global status-bar preferences.
- Fixed a stale README clone URL in the WSL setup instructions.
- Documented cleanup for legacy mise `ubi:akitaonrails/ai-jail` installs.

### Added

- Recognized `grok` as a known agent command (#51).
- Documented masking host machine identifiers such as `/etc/machine-id`.

### Packaging

- Updated checked-in AUR packaging to `v1.4.0` and documented that the project owns the live AUR package bases.
