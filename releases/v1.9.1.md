# v1.9.1

## NixOS hosts compatibility

### Fixed

- Fixed sandbox startup on NixOS and other immutable distros where `/etc/hosts` is a symlink into `/nix/store` (#64).
- ai-jail now resolves the `/etc/hosts` target and, when it points under `/nix`, mounts `/nix` early in the base mount group before applying the private hosts override. This avoids relying on user `--map /nix` entries, which are intentionally applied later.

### Documentation

- Documented the NixOS `/etc/hosts` behavior in the README examples.
