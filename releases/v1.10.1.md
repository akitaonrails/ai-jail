# v1.10.1

## Release workflow recovery

### Fixed

- Updated the GitHub release workflow so an Apple notarization account/agreement failure is reported as a warning instead of blocking the release artifacts.
- Disabled matrix fail-fast so a macOS release-lane failure does not cancel the Linux artifact upload.

### Included changes

- Includes the `v1.10.0` feature set: Pi support, command-scoped global config, `--deny-path`, `--systemd-user`, root-map refusal, project auto-save isolation, and Thunderbird deny-list support.
