- Added writable Kiro state support by mounting `~/.kiro` and `~/.local/share/kiro-cli` read-write inside the sandbox (#73).

- Added focused regression coverage for `.kiro` and `kiro-cli` writable mounts (#73).

- Verified formatting, clippy, workspace all-target clippy, unit/integration tests, macOS target check, dependency audit, and package verification before release.
