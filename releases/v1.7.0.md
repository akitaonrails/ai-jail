# v1.7.0

## Terminal proxy and AUR build fixes

### Fixed

- Fixed OpenCode rendering and terminal tab-title behavior under the PTY/status-bar proxy (#57).
- Restricted OSC 10/11/12 forwarding to query forms so sandboxed programs can query terminal colors without being allowed to set host terminal colors.
- Fixed the source AUR package build on environments where `check()` changes `HOME`: `CARGO_HOME` is pinned before overriding `HOME`, and `CARGO_TARGET_DIR` matches the build phase (#56).

### Note

- A `Release v1.6.0` version-bump commit exists in history, but no `v1.6.0` tag or GitHub release was published. These changes shipped in `v1.7.0`.
